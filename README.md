# Smartbridge Salesforce AI Assistant

## Objective
Build an intelligent chatbot that answers questions about Smartbridge Salesforce AI capabilities.

## Architecture

Smartbridge Video
↓
Transcript Extraction
↓
transcript.txt
↓
Python Chatbot
↓
Ollama (Llama 3)
↓
User Response

## Technologies Used

- Python
- Ollama
- Llama 3
- VS Code

## Features

- Transcript-based chatbot
- Conversational question answering
- Local LLM execution
- No Azure dependency

## Sample Questions

- What are some AI capabilities in Sales and Service?
- How is AI delivered across industry?
- Why can we trust Salesforce Generative AI?
- What are some mentioned use cases with Einstein Copilot?
